= Jekyll

== Installation sur ubuntu 22.10

[source,bash]
----
gem install jekyll bundler
----

TIP: Bundler est un gestionnaire de dépendances pour Jekyll

=== obtenir la version

[source,bash]
----
jekyll -v
----

NOTE: jekyll 4.3.2

== Creation du dossier pour le futur site

mkdir ~/wiki-yishan
